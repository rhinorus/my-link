2. УСЛОВИЯ ИСПОЛЬЗОВАНИЯ

2.1. Сервис позволяет создавать короткие ссылки, перенаправляющие на указанные Пользователем URL-адреса.

2.2. Особенности аутентификации:

  • Для доступа к расширенным функциям Сервиса может использоваться минимальная аутентификация через Telegram ID;
  • Telegram ID не является персональными данными согласно ФЗ-152.

2.3. Запрещено:

  • Создавать ссылки на запрещенный законодательством РФ контент;
  • Использовать Сервис для мошенничества или распространения вредоносного ПО;
  • Нарушать права третьих лиц.

2.4. Администрация вправе блокировать ссылки и аккаунты без предупреждения.
